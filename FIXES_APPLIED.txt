## Unified Contract

All handler return paths now follow this structure:

```python
{
    "valid": bool,                          # Required
    "session": ApplicationSession | None,   # Required
    "ready_for_execution": bool,            # Required
    "reason": str | None,                   # Required
    "workflow": str,                        # Required
    "execution_strategy": str,              # Optional (when applicable)
    "confidence": float,                    # Optional (when applicable)
    "page_analysis": PageAnalysisResult | None,  # Required (can be None)
    "execution_plan": ExecutionPlan | None,     # Required (can be None)
}
```
