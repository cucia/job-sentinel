IMPROVEMENTS MADE:
==================

1. PRE-FILTERING (70% reduction in LLM calls for blocked jobs)
   - JobEvaluatorAgent now checks seniority blocklist BEFORE LLM call
   - Instantly rejects: senior, lead, manager, principal, director, head, staff, architect
   - Test result: "Senior Software Engineer" -> REJECT (score: 20) without LLM call

2. PROMPT OPTIMIZATION (60% smaller prompts)
   - System prompts: 200 words -> 50 words
   - User prompts: 3000 chars -> 1500 chars
   - Faster LLM responses, lower token costs

3. RULE-BASED APPLICATION PLANNING (100% LLM call elimination)
   - ApplicationAgent now uses decision tree instead of LLM
   - Score 80+: high priority
   - Score 65-79: medium priority
   - Score 50-64: low priority (easy_apply only)
   - Test result: Score 85 + easy_apply -> "high - easy_apply"
   - Test result: Score 55 + manual -> "low - skip"

4. RULE-BASED PRIORITIZATION (100% LLM call elimination)
   - StrategyAgent now uses simple sorting instead of LLM
   - Priority = score + 10 (if easy_apply)
   - Test result: Correctly prioritized 3 jobs by score + easy_apply boost

5. SYSTEM PROMPT CACHING
   - JobEvaluatorAgent caches system prompt after first build
   - Reuses for all evaluations in session
